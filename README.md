## How To Use

Clone this repository
```bash
$ git clone https://github.com/DBDoco/ICE-CSS-Dummy-page.git
```

Go into the repository
```bash
$ cd ICE-CSS-Dummy-page
```

Install dependencies
```bash
$ npm i
```

Compile
```bash
$ gulp
```

Open index.html or add new html files.
